TROUBLESHOOTING:

Problem: "Network request failed"
Solution: Backend server not running or not reachable
Fix: Run START_BACKEND_SERVER.bat and verify step 2 above

Problem: "Registration failed"
Solution: Backend not accessible from phone
Fix: Check both devices on same WiFi, test http://10.126.93.209:8080/docs

Problem: Blank page after login
Solution: Metro bundler needs restart
Fix: Press Ctrl+C, then run: npm start

Problem: PDF download error
Solution: Libraries not installed
Fix: Run INSTALL_PDF_SIMPLE.bat
